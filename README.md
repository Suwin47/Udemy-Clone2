Udemy Clone Website 🎓
A responsive Udemy Clone Website built using HTML and CSS. This project replicates the basic layout and design of the Udemy homepage, including course cards, recommended topics, and footer section.

🚀 Features
Clean and modern homepage design
Course cards with images, ratings, and pricing
Recommended topics section
Responsive footer layout
Built using only HTML & CSS
🛠️ Technologies Used
HTML5
CSS3
📌 Purpose
This project was created to improve my front-end development skills by practicing layouts, flexbox, grid, and responsive design.
